I have created CrisisMap around these days.
What is CrisisMap:
I have created a platform where user will submit a form, form consists of location, emergency, name and number.
That details go as a report to departments.
OpenAI Nano Model score the report severity out of 5 based on the emergency message.
Initially the plan was that a map built from leatleaf.js will automatically drop different coloured pins based on score which has been executed, but the coordinates are hardcoded.

#How to use this 
Open the terminal,
clone this repository
after cloning, run python main.py.
